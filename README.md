# Packages pool
2026_robot 강의에 사용할 패키지들을 업로드하는 저장소입니다.

각 Author들의 패키지는 Branch로 구분하였으며, `git clone` 시에는 `-b {Author name}`을 최종 경로 뒤에 추가하여 사용합니다.

사용법은 추후 정리하여 공지하겠습니다.
